Goob18 YouTube TV Version List

list of TV Versions/Variants
  • Goob18 YouTube TV 2013 (default) - Initial release with basic features.
  • Goob18 YouTube TV MEK2013 (alt) - Alternate version that is compiled before 2014 and during late 2013.
  • Goob18 YouTube TV 2014 (broken) - Archived on June 11 2014 by the Wayback Machine.

How To Set Version

To set variant, press Alt+V. a prompt should pop up.
Version Codes
    2014
  • mek2013
  • 2013 (default)
if the variant isn't defined it will go to the 2013 variant
