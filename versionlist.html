Goob18 YouTube TV Version List

list of TV Versions/Variants
  • Goob18 YouTube TV 2013 (default) - Initial release with basic features.
  • Goob18 YouTube TV MEK2013 (alt) - Alternate version that is compiled before 2014 and during late 2013.
  • Goob18 YouTube TV 2014 (mostly fixed but with small bugs) - Archived on June 11 2014 by the Wayback Machine.
  • Goob18 YouTube TV Early 2013 - Archived on March 11 2013 at 1:25:49 PM by the Wayback Machine.
  • Goob18 YouTube TV 2013 (wayback machine variant) - same as default but images are now oginternet.
  • Goob18 YouTube TV 2012 (from wayback machine, feeds do not work) - wega curse warning.
  • Goob18 YouTube TV M14 (from wayback machine, normal feeds use gdata v2 but search is hardcoded to innertube).

How To Set Version

To set variant, press Alt+V. a prompt should pop up.
